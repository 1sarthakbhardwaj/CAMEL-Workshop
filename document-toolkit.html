📄 Document Toolkit

Process and extract content from PDF, Word, Excel, PowerPoint, images, and more

✨ Overview

The Document Toolkit in CAMEL-AI provides comprehensive document processing capabilities, combining the power of multiple toolkits including FileToolkit and MinerU Toolkit. Extract text, tables, formulas, and images from virtually any document format with OCR support.

📑 Supported Formats

📄 PDF Files

Extract text, tables, formulas with advanced OCR

📝 Word Documents

Process .doc and .docx files

📊 Excel Spreadsheets

Read and write .xls, .xlsx, .csv files

📊 PowerPoint

Extract content from .ppt and .pptx

🖼️ Images

OCR for .jpg, .jpeg, .png with text extraction

🌐 HTML

Parse and extract from HTML files

📚 EPUB

Extract content from e-books

📦 ZIP Archives

Process files within archives

🎵 Audio Files

Transcribe .mp3, .wav files

📋 Text Files

Process .txt, .json, .xml, .csv

🛠️ Main Toolkits

1. FileToolkit

General-purpose document processing with MarkItDown integration

from camel.toolkits import FileToolkit

toolkit = FileToolkit()

# Read single file
content = toolkit.read_file("document.pdf")

# Read multiple files
contents = toolkit.read_file([
    "doc1.pdf",
    "spreadsheet.xlsx",
    "presentation.pptx"
])

# Each returns markdown-formatted content
for path, text in contents.items():
    print(f"{path}:\n{text}\n")

2. MinerU Toolkit

Advanced document processing with OCR, formula, and table recognition

from camel.toolkits import MinerUToolkit

toolkit = MinerUToolkit(
    api_key="your_api_key",
    is_ocr=True,            # Enable OCR
    enable_formula=True,     # Recognize formulas
    enable_table=True,       # Detect tables
    language="en"
)

# Process from URL
result = toolkit.process_url(
    url="https://example.com/document.pdf",
    output_format="markdown"
)

# Process local file
result = toolkit.process_file(
    file_path="research_paper.pdf",
    output_format="markdown"
)

3. Excel Toolkit

Specialized Excel file operations

from camel.toolkits import ExcelToolkit

toolkit = ExcelToolkit()

# Read Excel file
df = toolkit.read_excel("data.xlsx", sheet_name="Sheet1")

# Write to Excel
toolkit.write_excel(
    data=df,
    file_path="output.xlsx",
    sheet_name="Results"
)

# Perform operations
summary = toolkit.analyze_data(df)

🚀 Quick Start Examples

Extract Text from PDF

from camel.toolkits import FileToolkit

toolkit = FileToolkit()

# Simple extraction
text = toolkit.read_file("document.pdf")
print(text)

# With OCR for scanned documents
from camel.toolkits import MinerUToolkit

ocr_toolkit = MinerUToolkit(is_ocr=True)
result = ocr_toolkit.process_file("scanned_doc.pdf")
print(result['markdown'])

Process Multiple Document Types

from camel.agents import ChatAgent
from camel.toolkits import FileToolkit

toolkit = FileToolkit()
agent = ChatAgent(
    system_message="You are a document processing assistant.",
    tools=toolkit.get_tools()
)

task = """
Read the following documents and create a summary:
1. report.pdf
2. data.xlsx
3. presentation.pptx
"""

response = agent.step(task)
print(response.msgs[0].content)

Extract Tables from PDF

from camel.toolkits import MinerUToolkit

toolkit = MinerUToolkit(
    enable_table=True,
    is_ocr=True
)

result = toolkit.process_file(
    "financial_report.pdf",
    output_format="markdown"
)

# Tables are preserved in markdown format
print(result['markdown'])

🔧 MinerU Configuration

MinerUToolkit(
    api_key="your_api_key",
    api_url="https://mineru.net/api/v4",
    
    # OCR settings
    is_ocr=False,           # Enable/disable OCR
    
    # Feature toggles
    enable_formula=False,   # Math formula recognition
    enable_table=True,      # Table detection
    
    # Model selection
    layout_model="doclayout_yolo",  # or "layoutlmv3"
    
    # Language
    language="en",          # or "zh" for Chinese
    
    # Processing
    wait=True,              # Wait for completion
    timeout=300             # Timeout in seconds
)

Limits

  • Maximum file size: 200MB per file
  • Maximum pages: 600 pages per file
  • Daily quota: 2000 pages for high-priority parsing

🎓 Best Practices

1. Choose the Right Tool

FileToolkit: General-purpose, fast, good for most documents

MinerU: Advanced OCR, formulas, complex layouts

ExcelToolkit: Specialized Excel operations

2. Enable OCR When Needed

Only enable OCR for scanned documents to save processing time

3. Process in Batches

Use batch processing for multiple files to improve efficiency

4. Handle Large Files

Split large PDFs before processing if they exceed limits
