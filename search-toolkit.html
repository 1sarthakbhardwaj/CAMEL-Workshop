Basic Usage with Brave

from camel.toolkits import SearchToolkit

toolkit = SearchToolkit()

# Search with Brave
results = toolkit.search_brave(
    q="What is CAMEL-AI?",
    search_lang="en",
    number_of_result_pages=10
)

print(results)

Using with ChatAgent

from camel.agents import ChatAgent
from camel.toolkits import FunctionTool, SearchToolkit

agent = ChatAgent(
    system_message="You are a helpful search assistant.",
    tools=[FunctionTool(SearchToolkit().search_brave)],
)

response = agent.step("What is the temperature in Tokyo?")
print(response.msgs[0].content)

Structured Output with Linkup

from pydantic import BaseModel

class PersonInfo(BaseModel):
    name: str = ""
    description: str = ""

    query="Nobel Prize winners in Physics",

    structured_output_schema=PersonInfo

Domain Exclusion

# Exclude specific domains from Google search
toolkit = SearchToolkit(
    exclude_domains=["wikipedia.org", "facebook.com"]
)

results = toolkit.search_google(
    query="machine learning tutorials"
)

ğŸ“ Best Practices

1. Choose the Right Engine

Google: Most comprehensive, best for general queries

Exa: Best for research papers and technical content

Brave: Good balance of quality and privacy

Chinese engines: Essential for Chinese language content

2. Optimize Result Count

Use fewer results for focused searches (3-5)

Use more results for comprehensive research (10-20)

3. Combine Multiple Engines

Use multiple search engines for cross-verification

4. Handle API Rate Limits

Implement retry logic and respect rate limits
