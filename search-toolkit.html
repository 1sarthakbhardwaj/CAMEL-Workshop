🔍 Search Toolkit

Unified interface to 12+ search engines for comprehensive web and academic research

✨ Overview

The Search Toolkit provides AI agents with powerful search capabilities across multiple platforms. From general web search to specialized academic databases, this toolkit offers a unified interface to gather information from diverse sources.

🚀 Quick Start

Basic Usage with Brave

from camel.toolkits import SearchToolkit

toolkit = SearchToolkit()

# Search with Brave
results = toolkit.search_brave(
    q="What is CAMEL-AI?",
    search_lang="en",
    number_of_result_pages=10
)

print(results)

Using with ChatAgent

from camel.agents import ChatAgent
from camel.toolkits import FunctionTool, SearchToolkit

agent = ChatAgent(
    system_message="You are a helpful search assistant.",
    tools=[FunctionTool(SearchToolkit().search_brave)],
)

response = agent.step("What is the temperature in Tokyo?")
print(response.msgs[0].content)

🛠️ Search Methods

Google Search

results = toolkit.search_google(
    query="artificial intelligence",
    search_type="web",  # or "image"
    number_of_result_pages=10,
    start_page=1
)

DuckDuckGo

results = toolkit.search_duckduckgo(
    query="machine learning",
    source="text",  # or "images", "videos"
    number_of_result_pages=10
)

Exa Neural Search

results = toolkit.search_exa(
    query="quantum computing research papers",
    search_type="neural",  # or "keyword", "auto"
    category="research paper",
    use_autoprompt=True,
    text=True,  # Include full text
    number_of_result_pages=10
)

Linkup Search

results = toolkit.search_linkup(
    query="Women Nobel Prize winners in Physics",
    depth="deep",  # or "standard"
    output_type="sourcedAnswer"  # or "searchResults", "structured"
)

Wikipedia

summary = toolkit.search_wiki(entity="Artificial Intelligence")

Chinese Search Engines

# Alibaba Tongxiao (optimized for Chinese)
results = toolkit.search_alibaba_tongxiao(
    query="阿里巴巴2024年的esg报告",
    time_range="OneMonth",
    industry="finance",
    enable_rerank=True
)

# Baidu
results = toolkit.search_baidu(
    query="今天北京的天气如何",
    number_of_result_pages=10
)

# Metaso
results = toolkit.search_metaso(
    query="What is CAMEL-AI?",
    scope="webpage"
)

🎯 Advanced Features

Structured Output with Linkup

from pydantic import BaseModel

class PersonInfo(BaseModel):
    name: str = ""
    description: str = ""

results = toolkit.search_linkup(
    query="Nobel Prize winners in Physics",
    output_type="structured",
    structured_output_schema=PersonInfo
)

Domain Exclusion

# Exclude specific domains from Google search
toolkit = SearchToolkit(
    exclude_domains=["wikipedia.org", "facebook.com"]
)

results = toolkit.search_google(
    query="machine learning tutorials"
)

Time-based Filtering

# Brave Search with freshness filter
results = toolkit.search_brave(
    q="latest AI developments",
    freshness="pd",  # past day: 'pd', week: 'pw', month: 'pm'
)

🎓 Best Practices

1. Choose the Right Engine

Google: Most comprehensive, best for general queries

Exa: Best for research papers and technical content

Brave: Good balance of quality and privacy

Chinese engines: Essential for Chinese language content

2. Optimize Result Count

Use fewer results for focused searches (3-5)

Use more results for comprehensive research (10-20)

3. Combine Multiple Engines

Use multiple search engines for cross-verification

4. Handle API Rate Limits

Implement retry logic and respect rate limits
