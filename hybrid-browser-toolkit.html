‚ú® Overview

The Hybrid Browser Toolkit represents the cutting edge of browser automation for AI agents. It combines non-visual, DOM-based browser automation with visual, screenshot-based capabilities, enabling sophisticated web interactions that were previously impossible for AI agents.

From the Notion article: The toolkit bridges the gap between traditional automation and AI-powered agents by providing both programmatic control and visual understanding of web pages.

üöÄ Quick Start

Basic Usage

from camel.toolkits import HybridBrowserToolkit
from camel.agents import ChatAgent
from camel.models import ModelFactory
from camel.types import ModelPlatformType, ModelType

# Initialize the model
model = ModelFactory.create(
    model_platform=ModelPlatformType.OPENAI,
    model_type=ModelType.GPT_4O,
)

# Create the toolkit with custom tools
custom_tools = [
    "browser_open",
    "browser_close",
    "browser_visit_page",
    "browser_click",
    "browser_type",
    "browser_enter",
]

web_toolkit = HybridBrowserToolkit(
    headless=False,
    user_data_dir="User_Data",
    enabled_tools=custom_tools,
    browser_log_to_file=True,
    stealth=True,
    viewport_limit=True,
)

# Create an agent with the toolkit
agent = ChatAgent(
    model=model,
    tools=[*web_toolkit.get_tools()],
    toolkits_to_register_agent=[web_toolkit],
)

# Use the agent
task = """
Use Google Search to find news about AI, 
click on relevant websites, and summarize the content.
"""

response = await agent.astep(task)
print(response.msgs[0].content)

Advanced: Python vs TypeScript Mode

# TypeScript mode (default) - WebSocket-based
toolkit_ts = HybridBrowserToolkit(
    mode="typescript",
    headless=False,
    stealth=True,
)

# Python mode - Pure Playwright
toolkit_py = HybridBrowserToolkit(
    mode="python",
    headless=True,
    viewport_limit=True,
)

# Connect to existing browser via CDP (TypeScript only)
toolkit_cdp = HybridBrowserToolkit(
    mode="typescript",
    connect_over_cdp=True,
    cdp_url="ws://localhost:9222/devtools/browser/...",
    cdp_keep_current_page=True,
)

üõ†Ô∏è Available Tools

Core Navigation Tools

  • browser_open - Initialize a new browser session
  • browser_close - Close the browser session
  • browser_visit_page - Navigate to a URL
  • browser_back - Go back in history
  • browser_forward - Go forward in history

Interaction Tools

  • browser_click - Click on elements
  • browser_type - Type text into inputs
  • browser_select - Select dropdown options
  • browser_scroll - Scroll the page
  • browser_enter - Press Enter key
  • browser_press_key - Press any keyboard key
  • browser_mouse_control - Advanced mouse movements
  • browser_mouse_drag - Drag and drop operations

Information Gathering

  • browser_get_page_snapshot - Get page DOM snapshot
  • browser_get_som_screenshot - Get screenshot with marked elements
  • browser_get_page_links - Extract all links from page
  • browser_console_view - View console messages

Tab Management

  • browser_switch_tab - Switch between tabs
  • browser_close_tab - Close specific tab
  • browser_get_tab_info - Get tab information

Advanced Tools

  • browser_solve_task - AI-powered task solving
  • browser_console_exec - Execute JavaScript in console
  • browser_wait_user - Wait for user interaction

üîß Configuration Options

HybridBrowserToolkit(
    # Mode selection
    mode="typescript",  # or "python"
    
    # Browser settings
    headless=True,      # Run without GUI
    stealth=True,       # Enable anti-detection
    user_data_dir="User_Data",  # Persist cookies/sessions
    
    # Performance optimization
    viewport_limit=True,  # Only include visible elements
    cache_dir="tmp/",     # Cache directory
    
    # Timeouts (in milliseconds)
    default_timeout=30000,
    navigation_timeout=30000,
    screenshot_timeout=5000,
    
    # Logging
    browser_log_to_file=True,
    log_dir="browser_log",
    
    # CDP mode (TypeScript only)
    connect_over_cdp=False,
    cdp_url=None,
    cdp_keep_current_page=False,
    
    # Visual mode
    full_visual_mode=False,
    
    # Tools configuration
    enabled_tools=HybridBrowserToolkit.ALL_TOOLS,
)

üéì Best Practices

1. Choose the Right Mode

TypeScript mode: Better for complex interactions and CDP support

Python mode: Simpler setup, better for basic automation

2. Optimize Context Size

Enable viewport_limit=True to reduce context sent to LLMs

Remove unnecessary tools from enabled_tools

3. Use Stealth Mode Wisely

Enable stealth=True for sites with bot protection

Combine with user_data_dir for persistent sessions

4. Handle Errors Gracefully

Always clean up with await toolkit.browser_close()

Use try-finally blocks to ensure cleanup
